========================================
COMMON ISSUES & SOLUTIONS
========================================

Issue 1: public folder too large to upload
-------------------------------------------
Solution: 
  - ZIP it first (compresses to ~800 MB)
  - Upload ZIP
  - Extract on server
  - Much faster!

Issue 2: Upload stops midway
-----------------------------
Solution:
  - Check your internet connection
  - Try smaller chunks (upload assets/ separately)
  - Use FTP if available (faster)

Issue 3: Still blank after upload
----------------------------------
Solution:
  - Verify index.html exists in public/
  - Check assets/ folder has JS/CSS files
  - Check server logs for 404 errors
  - Clear browser cache completely

Issue 4: 404 Not Found errors
------------------------------
Solution:
  - Check server is serving from correct path
  - Verify startup file is server.cjs
  - Check index.js has serveStatic code
  - Restart app
